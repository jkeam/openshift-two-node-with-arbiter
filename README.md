# OpenShift Two Node With Arbiter

## Deployments

OpenShift Two Node with Arbiter.

Examples deployments can be found in `./openshift/`.

## Overhead

To see rough estimate overhead for OpenShift and KubeVirt, we have two scripts.

1. `./scripts/openshift-overhead.py` - OpenShift overhead
2. `./scripts/kubevirt-overhead.py` - KubeVirt overhead
